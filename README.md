- Responsive design for desktop, tablet, and mobile

- Animated skill progress bars

### Clone the repository

```bash

```

### Navigate into the project

```bash
cd portfolio
```

### Open the website

Simply open

```
index.html
```

in your browser.

No installation or dependencies are required.
